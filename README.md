# 📖 Overview

This project was completed as part of a **Data Engineering & Analytics Internship Assessment**.

The objective was to build an end-to-end analytics workflow using the **Boston Airbnb dataset**, covering data engineering, statistical analysis, SQL database implementation, and interactive business intelligence dashboards.

# ▶️ How to Run the Project

## 📒 Jupyter Notebook

### Install the required Python libraries

```bash
pip install pandas numpy matplotlib seaborn
```

### Open Jupyter Notebook

Launch Jupyter Notebook and open the notebooks in the following order:

1. 📓 Data_Cleaning.ipynb
2. 📓 data_validation.ipynb
3. 📓 Join_Data.ipynb
4. 📓 Feature_Engineering.ipynb
5. 📓 statistical_analysis.ipynb

Run each notebook from the first cell to the last cell before proceeding to the next notebook.

> **Note:** Ensure that the dataset paths are updated to match your local directory before running the notebooks.

---

## 🗄️ SQL Server

1. Create the database
2. Execute SQL scripts
3. Import cleaned datasets
4. Run analytical queries

---

## 📈 R

Install required packages

```R
install.packages("tidyverse")
install.packages("ggplot2")
install.packages("car")
install.packages("effectsize")
install.packages("lmtest")
install.packages("broom")
install.packages("emmeans")
```

Run:

```
hypothesis_testing.R
```

Open:

Refresh the data source if required.
